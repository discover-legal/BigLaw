The licenseExhibit B
AGPL → Apache-2.0

The most-requested change was never a feature.

Network copyleft is a hard conversation for a firm’s IT and legal review — AGPL kept capable teams from even piloting. Now: express patent grant, NOTICE attribution, no network-service source obligation. Getting there honestly took a clean room, not a paraphrase.

3
Independent implementers, attested
Clean parties built from the spec and the public OOXML standard alone, each signing a non-exposure attestation. Dated paper trail.

How · technique 1 of 3Exhibit J
Grounding

The model never types a digit.

¶
The substring-lock
Evidence is transcribed verbatim, verified as a literal substring of the source, and locked. Conclusions are written only over locked quotes — anything paraphrased is dropped before it can become a citation.
¶
Figure handles
Drafters drop a neutral codename into prose — the digit masked even from the prompt — and the exact grounded value is substituted afterward, by key. A model that never types a number can never garble one.
≈0% → 94%

verbatim-grounded citations on a local model — from roughly zero.

How · technique 2 of 3Exhibit K
Extraction floor

The funnel was the ceiling.

The pipeline was feeding a 200K-context model through a straw sized for a 4K local one — evidence caps, passage limits, and truncation all wearing local-model-era defaults. The criterion-level autopsy attributed the entire gap to Harvey’s own harness to that funnel.

§1Saturation harvest a deterministic walk of every section of every document — nothing left to retrieval luck
§2Full-text ingest document reads flow into the evidence pool untruncated
§3Context-aware caps evidence and tool-result budgets auto-scale with the model’s window instead of hard-coded limits
34 → 49 of 60 — the extraction floor behind the fix wave.

How · technique 3 of 3Exhibit L
Mechanical honesty

The system voids its own bad runs.

§
Cross-document joins
Figures are joined across the corpus mechanically — two documents that disagree on a number or a date surface together, both verbatim quotes attached.
§
Defense lenses
An extraction memo for the defense also says what the defense can do about each allegation — exposure, mitigants, procedural posture — on a deterministic trigger.
§
Starvation guards
A round where every agent came back empty marks the run starved — a dead run can no longer silently score. Timed-out agents get one retry at an extended budget.
The run history holds a voided 27/60 — forensics showed every round had timed out under contention.
The guards exist so the next one voids itself, before it can enter the record.

Operator surfaceExhibit M
The knobs

Tuned for a Pi by default.
Unlocked for the cloud.

⊕
Context-aware caps
Evidence and tool-result budgets size themselves to the model’s context window — small for a local 4K model, wide for a 128K cloud one. Override: MODEL_CONTEXT_TOKENS, the AGENT_MAX_* family.
↻
Timeout retry
An agent that blows the round budget gets one retry at an extended budget — slow local models stop tripping the round clock. ROUND_TIMEOUT_RETRY_FACTOR
⊘
Task quarantine
Tasks caught mid-run at boot are quarantined as interrupted — audited, closed out, never resumed as phantoms contending for the model. RESUME_RUNNING_TASKS

Every default answers to the smallest machine the platform ships for; every ceiling opens with one variable.
